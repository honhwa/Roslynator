## Inline constant value

| Property           | Value                              |
| ------------------ | ---------------------------------- |
| Id                 | RR0127                             |

| Syntax             | expression that has constant value |
| Enabled by Default | &#x2713;                           |

### Usage

#### Before

```csharp
public const string Value = "x";

void Foo()
{

}
```

void Foo()
{

}

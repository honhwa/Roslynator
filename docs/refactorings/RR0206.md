## Replace foreach with enumerator

| Property           | Value                           |
| ------------------ | ------------------------------- |
| Id                 | RR0206                          |
| Title              | Replace foreach with enumerator |
| Syntax             | foreach statement               |
| Span               | foreach keyword                 |
| Enabled by Default | &#x2713;                        |

### Usage

#### Before

```csharp
foreach (var item in items)
{
    yield return item;
}
```

#### After

```csharp
using (var en = items.GetEnumerator())
{
    while (en.MoveNext())
    {
        yield return item;
    }
}
```
